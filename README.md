# Portfólio Pessoal (Base Inicial)

Este projeto agora está preparado como ponto de partida para um portfólio pessoal.

## Estrutura atual

- `index.html`: página única com seções de apresentação, sobre, projetos e contato.
- `estilos/style.css`: estilos responsivos e organizados com variáveis CSS.
- `imagens/`: pasta para assets visuais.

## Próximos passos recomendados

1. Substituir "Seu Nome" e textos de exemplo por informações reais.
2. Atualizar links de GitHub, LinkedIn e e-mail.
3. Trocar os cards de projeto por projetos reais (com links e imagens).
